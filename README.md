# CS-370-Portfolio-Reflection

Project Overview

In a Jupyter Notebook, I used reinforcement learning techniques to create an intelligent pirate agent for this project. The objective was to teach the agent how to find the treasure while avoiding obstacles in a maze. Through this project, I was able to develop an agent that made better decisions over time by applying ideas like neural networks, Q-learning, exploration versus exploitation, and experience replay.

The project framework, which includes the environment setup, maze configuration, assistance functions, and a large portion of the supporting code required to complete the project, was supplied by the teacher. The components of reinforcement learning that enabled the pirate agent to learn from its experiences were implemented by myself. I finished the training loop, used an exploration strategy to choose activities, managed experience replay, updated the neural network based on rewards, and changed parameters to enhance the agent's performance until it consistently located the treasure.

What Do Computer Scientists Do and Why Does It Matter?

Through algorithm design, software development, data analysis, and the creation of systems that enhance productivity and decision-making, computer scientists solve complicated challenges. Almost every industry is impacted by their work, including cybersecurity, healthcare, education, banking, transportation, and entertainment. Computer scientists assist businesses in automating processes, increasing productivity, and resolving practical problems by developing dependable and intelligent software solutions.

How Do I Approach a Problem as a Computer Scientist?

As a computer scientist, I tackle issues by first comprehending the specifications and dividing complex issues into smaller, more manageable chunks. I evaluate the information at hand, find potential fixes, regularly test my code, and adjust my strategy in response to the outcomes. This course taught me the value of testing, troubleshooting, and assessing machine learning model performance. I discovered that enhancing an intelligent agent necessitates ongoing testing and modification rather than anticipating a quick fix.

What Are My Ethical Responsibilities to the End User and the Organization?

It is my duty as a computer scientist to create software that is safe, dependable, and equitable. End users should be able to rely on the systems they use to safeguard their data and function as intended. Additionally, companies depend on developers to build precise, manageable, and thoroughly documented code that adheres to ethical standards and reduces security threats. Transparency, fairness, privacy, and the possible effects of automated judgments on people are particularly crucial when dealing with AI. Adhering to moral standards promotes trust and guarantees that technology is advantageous to businesses and society as a whole.
